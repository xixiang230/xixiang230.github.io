学习背景

从今天开始Docker源码的学习和总结，以《Docker源码分析》一书的结构和解说为基础，配合其它资料学习，旨在弄懂每一个细节和概念。

Docker介绍

Docker借助操作系统层的虚拟化来实现资源隔离，因此在运行时，Docker容器与宿主机是共享同一操作系统的，不会产生额外的操作系统开销，从而可大大提高资源利用率和提升I/O性能。
